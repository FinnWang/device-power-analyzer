## 🚀 使用方式

### 快速開始

## 🔧 技術架構

### 核心技術棧
- **資料處理**: pandas + numpy
- **視覺化**: matplotlib + seaborn + plotly
- **GUI框架**: tkinter + streamlit
- **測試框架**: pytest
